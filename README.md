# xbps-gtk

XBPS package manager GUI built with GTK4.

## Requirements

- GTK4 >= 4.10.0
- libxbps
- gettext

## Build

```bash
./autogen.sh
./configure
make
```

## Features

- Package browser and search
- Repository management
- Transaction handling (install/remove/upgrade)
- Graphical user interface

## Usage

```bash
./xbps-gtk
```
